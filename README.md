## Usage

### clone

```sh
git clone https://github.com/sankeyangshu/react-calculator.git
cd react-calculator
```

### install

```sh
pnpm install
```

### Development

Just run and visit http://localhost:5173

```sh
pnpm dev
```

### Build

To build the App, run

```sh
pnpm build
```

And you will see the generated file in dist that ready to be served.
